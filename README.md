# Formulário de Convite

Projeto desenvolvido com HTML5 e CSS3 que consiste em um formulário para criação de convites de eventos. O objetivo foi praticar a construção de formulários modernos, estilização de campos personalizados e organização de layouts utilizando boas práticas de desenvolvimento front-end.

## ✨ Funcionalidades

- Cadastro de informações do evento
- Seleção do tipo de evento
- Escolha entre evento presencial ou online
- Seleção de tema do convite
- Upload de imagem de capa
- Campos de contato do organizador
- Validação de campos obrigatórios pelo HTML
- Interface moderna e responsiva

## 📚 Conceitos praticados

Durante o desenvolvimento deste projeto foram aplicados conceitos como:

- Estrutura semântica em HTML
- Criação de formulários completos
- Inputs personalizados
- Estilização de botões, switches e campos de seleção
- Pseudo-classes (`:hover`, `:focus`, `:checked`, `:has()`)
- Flexbox
- CSS Grid
- Variáveis CSS
- Organização de arquivos
- Responsividade

## 🎯 Objetivo

Este projeto foi desenvolvido para praticar a criação de formulários modernos utilizando apenas HTML e CSS, explorando diferentes tipos de campos, componentes personalizados e técnicas de estilização sem a utilização de JavaScript.
